# Инструкция по размещению изображений

## Структура папок

```
youtalk/
├── img/          # Фотографии для статей
├── icons/        # SVG иконки для тегов
└── index.html
```

## Иконки для тегов

В папке `icons/` уже созданы SVG иконки:
- `tag-self.svg` - иконка для тега "Self"
- `tag-smile.svg` - иконка для тега "Все"
- `tag-unicorn.svg` - универсальная иконка единорога

## Замена placeholder-изображений

После добавления реальных изображений замените в `index.html`:

**Было:**
```html
<img src="https://placehold.co/588x260/..." alt="...">
```

**Стало:**
```html
<img src="img/article-1.jpg" alt="...">
```

## Рекомендуемые размеры изображений

- **Большие карточки**: 588x260px (соотношение ~2.26:1)
- **Обычные карточки**: 384x260px (соотношение ~1.48:1)
- **Иконки тегов**: 20x20px (SVG)

## Оптимизация

Рекомендуется оптимизировать изображения перед размещением:
- Используйте формат WebP для лучшего сжатия
- Оптимизируйте JPEG с качеством 80-85%
- Используйте инструменты типа TinyPNG или ImageOptim
